Turn off JavaScript

Now…to server-side rendering. Try turning off javascript in your web browser and refreshing the page. You’ll notice that you get the exact same result! That is because we render the same page on the server before it is sent to the user. This makes it so that your React applications can be fully search engine optimized. Now turn JavaScript back on and refresh the browser so you can continue building your app.
